[![React](https://img.shields.io/badge/React-18.x-61dafb.svg)](https://reactjs.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-3.x-38bdf8.svg)](https://tailwindcss.com/)

An open-source web-based system monitor for visualizing and tracking real-time performance metrics including CPU usage, memory consumption, disk usage, network activity, and running processes.

## Overview

Understanding system performance is crucial for identifying bottlenecks and optimizing productivity. The OS Productivity Dashboard provides a modern, responsive interface for monitoring your system resources in real-time. Built with React and styled with Tailwind CSS, this dashboard offers an intuitive way to keep track of your system's health and performance.

## Features

### Core Monitoring Capabilities

- **CPU Monitoring** - Track overall CPU usage, per-core statistics, and temperature readings
- **Memory Usage Visualization** - Real-time memory consumption tracking with visual indicators
- **Disk Usage Monitoring** - Monitor storage capacity and usage patterns
- **Network Activity Tracking** - Visualize network traffic and bandwidth utilization
- **Process Monitoring** - View running processes with detailed status indicators
- **System Information Overview** - Comprehensive system specs and information at a glance

### Design & User Experience

- Modern responsive dark UI optimized for readability
- Real-time data visualization with interactive charts
- Clean and intuitive navigation between different monitoring views

## Installation

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/allubhujia/OS_Productivity_Dashboard
   cd os-productivity-dashboard
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` (or the port shown in your terminal)

## Usage

The dashboard provides multiple views for different monitoring needs:

- **Dashboard Overview** - Navigate to `/dashboard` for a comprehensive system overview
- **CPU Monitoring** - Navigate to `/cpu` for detailed CPU performance metrics
- Additional views for memory, disk, network, and process monitoring
